# Students Table – React Project

This is a small React project that displays and manages a list of students in a table.
All operations like add, edit, and delete are handled on the frontend using React state.

This project was created as a frontend assignment.

---

## Features

* View list of students in a table
* Add a new student
* Edit student details
* Delete student with confirmation
* Form validation (all fields required, valid email)
* Simulated loading state
* Search students by name
* Download student data in Excel format

---

## Tech Used

* React
* Tailwind CSS
* JavaScript
* XLSX (for Excel download)

## How to Run the Project

1. Clone the repository

git clone https://github.com/yourusername/students-table.git

2. Go to the project folder

cd students-table

3. Install dependencies

npm install

4. Start the development server

npm run dev

5. Open in browser

http://localhost:5173

---

## Note

This project works completely on the frontend.
Student data is stored in the application state, so the data will reset when the page is refreshed.
